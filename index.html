Canvas

Canvas is a Java application that can be used by universities and schools to assign, view and grade assignments or homework specifically for programming courses.
Features include Bulk Compilation; Code and Output Display; Automatic Class Grading;
Student and Class Performance Calculation; Graphical Performance Representation etc.

EasyPostscript

EasyPostscript is a dynamically typed vector graphics tool developed using C++ and Postscript.
It can be used for drawing designs ranging from simple ones like geometrical shapes to
complex designs like bar graphs, line graphs, charts or even the famous Mandelbrot set.

PySport

PySport is a Python application that predicts the result of sports matches based on tweets made by fans, commentators, sporting authorities and
journalists throughout the week before the match. Several Python libraries are used including tweepy, matplotlib, numpy and
numpy that look for patterns, trends and keywords to make a prediction and display the results graphically.
